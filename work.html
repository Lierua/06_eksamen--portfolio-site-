HOME SITE OF 03_UX/UI

THE START OF PROJECTS

I was introduced to the importance of brainstorming and learned how to sketch out ideas effectively. This allowed me to explore different creative directions and refine concepts before starting detailed work.


Doing this period I also got to learn various research methods, such as desk research and questionnaire research.

MOODBOARD

Learned about moodboards and how they help set the visual tone for a project by combining images, colors, and typography. and how to use it to translate ideas into a cohesive style.

STYLETILES

A style tile showcases the key visual elements of a design, like colors, fonts, and UI components. I learned how to use it to ensure consistency and align the visual identity with the project's goals.

USE OF FIGMA FOR PROJECT

Although we had been introduced to Figma before, this theme really focused on using it extensively. I learned how to create detailed wireframes, progressing from low-fidelity sketches to high-fidelity designs, and even built interactive prototypes to simulate user experiences.


Figma has also become my go to for organising web projects, as seen for the projects 03_UX/UI, 05_VIDEO and for this exam project

LOFI-WIREFRAMES

Lo-fi wireframes are simple sketches or digital layouts that focus on structure and functionality. I used them to map out the basic layout and user flow before adding visual details.

HIFI-WIREFRAMES

Hi-fi wireframes are detailed and polished designs that include colors, typography, and UI elements. They allowed me to refine the visual style and prepare for development.

PROTOTYPES

Prototypes bring designs to life by simulating interactions and user flows. I learned how to create clickable prototypes to test usability and gather feedback before final implementation.

TESTS

I was introduced to various usability testing methods, such as the 5-second test to gauge first impressions, the think-out-loud and Likert test for gathering user feedback, and the Lighthouse test for assessing website performance and accessibility.

PRESENTATION OF PROJECT

We concluded the theme with a presentation where we showcased the websites we had created, explaining our design choices and development process.
